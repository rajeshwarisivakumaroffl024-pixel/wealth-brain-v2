Step-by-step: How to file ITR-1

1. Go to incometax.gov.in → Login with PAN

2. e-File → Income Tax Returns → File ITR → AY 2026-27

3. Select ITR-1 (Sahaj) → your info auto-fills from PAN

4. Enter salary from Form 16 (ask Adhithya Holidays)

5. Enter bank interest from all 3 bank statements

6. Add SIP/80C deductions if any

7. System computes tax → verify with Aadhaar OTP

8. Submit → save acknowledgement number
